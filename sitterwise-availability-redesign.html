Design Notes for Aji

  • Area icons — Hotel Del icon = South County, Balloon icon = North County. Dark fill = preferred area, light fill = will go there but not preferred
  • Color palette — Navy (#1B3A5C) as the dominant accent, teal (#84D0D2, #E8F5F5) for backgrounds and borders, coral (#F48A91) reserved for primary action buttons and the booked indicator only
  • Typography — Playfair Display for headings, Poppins for all UI text — matching sitterwise.com
  • Buttons — Square corners (border-radius: 0) per brand standard; coral primary, teal-outline secondary
  • Filters — Pill/chip style replaces plain checkboxes for a cleaner, more tappable feel
  • Time-of-day chips — Color-coded: gold = morning, teal = afternoon, purple = evening (matches availability dots in results)
  • Availability dots — Replace the starburst icons with clean circles using M/P/E letters and color coding for quick scanning
  • Booked status — Coral dot with "B" distinguishes booked from blocked/unavailable
  • Table header — Navy background with white text for strong visual anchor and easy scanning
  • Hover state — Blush (#FDF5F2) row highlight on hover
  • Expertise tags — Color-coded mini-pills replace icon-only system for better readability
